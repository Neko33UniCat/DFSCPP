# DFS-C++

## 深さ優先探索(だと思う)

### DFS
`$ git clone https://gitlab.com/UniCatHMR/dfs-cpp.git`

`$ cd DFS`

* Mac
    * `$ g++ -std=c++17 DFS.cpp -o DFS`
    * `$ ./DFS < input (or input2)`
    * `$ 9`
* Windows
    * `$ g++ -std=c++17 DFS.cpp -o DFS.exe`
    * `$ .¥DFS < input (or input2)`
    * `$ 9`

#### 標準入力

> 親nodeの番号(root node は入力しない)	子nodeに伸びている枝の数 子nodeの番号(子nodeに伸びている枝の数==0の時は自身の値)
